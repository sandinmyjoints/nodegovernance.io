On February 10, Joyent announced that control of Node.js would move to a foundation. Important details about how the foundation will work have not yet been decided.

Now is the time to tell Joyent's CEO Scott Hammond that as a Node user and community member, you would like io.js's open technical governance model to be the basis for the Node.js foundation's technical committee.
